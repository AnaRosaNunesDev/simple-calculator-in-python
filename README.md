# 🧮 Calculadora em Python com Tkinter

Uma calculadora simples, moderna e totalmente funcional desenvolvida em Python. O projeto utiliza uma interface gráfica estilizada para realizar operações matemáticas básicas em tempo real.

---

## 🚀 Funcionalidades

- **Operações Básicas:** Soma (`+`), Subtração (`-`), Multiplicação (`*`) e Divisão (`/`).
- **Porcentagem:** Cálculo rápido usando o operador `%`.
- **Tratamento de Erros:** Exibe uma mensagem de "Erro" na tela caso o usuário digite uma sequência matemática inválida (como múltiplos operadores juntos).
- **Limpeza de Tela:** Botão `C` para zerar o visor e reiniciar os cálculos rapidamente.

---

## 🛠️ Tecnologias Utilizadas

- **Python 3** (Linguagem base)
- **Tkinter / TTK** (Biblioteca nativa para construção da interface gráfica)

---

## 🎨 Design e Visual

O visual da aplicação foi planejado com uma paleta de cores moderna e escura (Dark Mode):
- **Fundo do visor:** Azul carregado (`#141847`) com textos em branco.
- **Botões numéricos:** Cinza clássico com fontes em negrito.
- **Botões de operação:** Destaque personalizado na cor Rosa/Pink (`#B965B5`).

---

## 📦 Como Executar o Projeto

Certifique-se de ter o Python instalado na sua máquina. Não é necessário instalar nenhuma biblioteca externa, pois o `tkinter` já vem integrado por padrão.

1. Clone o repositório ou baixe os arquivos.
2. Abra o terminal na pasta do projeto.
3. Execute o comando:
   ```bash
   python main.py
   ```
